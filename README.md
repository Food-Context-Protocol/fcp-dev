# fcp.dev

Landing page for the [Food Context Protocol](https://github.com/Food-Context-Protocol/fcp).

Built with [Astro](https://astro.build) and deployed to Cloudflare Pages.

## Development

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # Build to ./dist/
```

## Related Repositories

- [fcp](https://github.com/Food-Context-Protocol/fcp) - Protocol specification
- [fcp-gemini-server](https://github.com/Food-Context-Protocol/fcp-gemini-server) - Reference server implementation
- [fcp-cli](https://github.com/Food-Context-Protocol/fcp-cli) - Command-line interface
